# CSC415 - Introduction to Reinforcement Learning

Course webpage for the Winter 2026 course on Introduction to Reinforcement Learning (CSC415) at University of Toronto Mississauga.

## Website

The course website is available at: https://utm-csc415.github.io/w2026/

## Local Development

To run the site locally:

1. Install Ruby and Bundler
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Run Jekyll:
   ```bash
   bundle exec jekyll serve
   ```
4. Open http://localhost:4000/w2026/ in your browser

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the `main` branch.
